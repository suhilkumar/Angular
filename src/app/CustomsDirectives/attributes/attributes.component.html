What is Attribute Directive

It will change the appearance or behavior of Dom Elements and Angular components

How to create Custom Attribute Directive

create a custom Directive ng generate directive "directiveName"

  1. we need to import the element ref from angular core

    2. What is elementRef

    It grants direct access to the host dom elements through nativeElement Property

    we do it with di you inject in constructor of your directives

  2. Set the color to yellow with elementRef
    3. this.elementRef.nativeElement.style.backgroundColor = "yellow"
  3. now set the directive in the element and import it in its component.ts file
  4. You can also apply the event in the directives as well with the help of HostListener("event")
  5. You can also bind send data with the help of input binding but make sure to name it as directive name and use property binding syntax

    6. [directiveName]="variable"

  6. whenever you want expression in html you use NgNonBindable

    7. with ngNonBindable

    {{ 1 + 1 }}

    without ngNonBindable

    {{ 1 + 1 }}
